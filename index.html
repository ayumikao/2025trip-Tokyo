使用說明

  1. 上傳到 GitHub 倉庫根目錄（檔名 index.html）。
  2. 開啟 Pages：Settings → Pages → Branch: main／Folder: / (root)。
  3. 等約 1 分鐘，開啟 https://你的帳號.github.io/你的倉庫/。

※ 需要獨立相簿/更多分頁，可加檔 photos.html、timeline.html 並在導覽列連結。
